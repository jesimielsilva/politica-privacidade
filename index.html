Política de Privacidade

Última atualização: 06/05/2025

Este aplicativo (“Motivacional”) respeita sua privacidade. Esta Política de Privacidade descreve como coletamos, usamos e protegemos suas informações ao utilizar nosso aplicativo.

1. Coleta de Dados

O app não coleta dados pessoais sensíveis como nome, e-mail, telefone, fotos, etc.

Porém, alguns dados anônimos e técnicos podem ser coletados automaticamente por serviços de terceiros:

1.1 Google AdMob

  • Coleta identificadores do dispositivo
  • Informações de localização aproximada (via IP)
  • Interações com anúncios

Esses dados são usados para exibir anúncios personalizados ou não personalizados.

1.2 Firebase Analytics

  • Coleta dados como tempo de uso, telas mais acessadas e cliques

Esses dados são anônimos e utilizados para melhorar a experiência do usuário.

2. Uso dos Dados

Utilizamos os dados coletados para:

  • Melhorar o desempenho do app
  • Exibir anúncios relevantes
  • Compreender como os usuários interagem com o aplicativo

3. Compartilhamento de Dados

Não compartilhamos suas informações pessoais com terceiros, exceto:

  • Quando exigido por lei
  • Com serviços como Google AdMob e Firebase, conforme suas políticas

4. Segurança

Aplicamos práticas modernas de segurança para proteger seus dados contra acessos não autorizados.

5. Consentimento

Ao utilizar o Motivacional, você concorda com esta Política de Privacidade.

6. Alterações nesta Política

Podemos atualizar esta política a qualquer momento. A data da última modificação estará sempre indicada acima.
